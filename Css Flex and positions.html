CSS Essentials - Flex and Positions

CSS Flexbox

Flexbox is a layout model which helps the elements to behave according to the different screen sizes and displays. The main purpose of Flexbox is to provide an efficient way to layout and align the items. CSS Flexbox contains two components :

  • Flexbox Container
  • Flex Item
Flexbox Container

Flexbox Container provides the parent properties.

Flexbox Items

Flexbox items provides the children properties.

Following is an example to show four flex items within a flex container:

Display flex

Display flex is a property of flexbox container. By using display flex to parent container, the children automatically enable flex content. By using display flex we can easily align the items in center.

CSS Positions

The CSS position property is used to set position for an element. There are three positioning in CSS:

  • Relative positioning
  • Absolute positioning
  • Fixed positioning

CSS Z-Index

It is used to align the elements on 3-dimensional plane,i.e., z-axis. It only works when the position of elements are absolute, fixed or relative.






In the above example, the z-indexof blue box is 3, so it has appeared on top of green and red box.

Z-index property affects the stacking order of positioned elements. Whichever element has more z-index will appear at top on the stack.







In the above example, z-index of green box is 3, so it appears above of other two boxes. z-index of blue is 2 and red is 1, so blue appears above the red box and red appears at last.
